# Data Directory

This directory contains the textual corpora used in the project.

## raw/
Raw, unprocessed texts in plain `.txt` format.

Expected files:
- odyssey.txt
- ulysses.txt
- eneida.txt

Due to size and licensing considerations, the corpus files are not included in the repository. All texts are publicly available.

## processed/
Processed and cleaned versions of the texts generated during analysis.
(This directory is created during runtime.)
